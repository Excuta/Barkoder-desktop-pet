# Security Policy

## Scope

Barkoder is a personal Windows desktop-pet application. It runs locally, makes no network requests, and stores only session log files in `~/.barkoder/`.

## Reporting a Vulnerability

If you discover a security issue, please open a [GitHub Issue](../../issues) with the label `security`. There is no formal SLA — this is a personal project — but reports will be reviewed promptly.

Please do not report vulnerabilities via public pull requests until a fix is available.
